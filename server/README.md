# Smart Road Maintenance System Backend

This is the backend for the Smart Road Maintenance System MERN stack project.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- Multer

## Setup

Install dependencies:

```bash
npm install
```

Run in development mode:

```bash
npm run dev
```

Run in production mode:

```bash
npm start
```

## Environment Variables

Update `.env` with your own values:

```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/smart-road-maintenance
JWT_SECRET=replace_with_a_secure_jwt_secret
CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_API_SECRET=your_cloudinary_api_secret
```
